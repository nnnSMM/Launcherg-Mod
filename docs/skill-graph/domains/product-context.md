# Product Context

Launcherg-Modは、Windows上のゲームを登録、起動、整理し、プレイ時間、メモ、スクリーンショットをまとめて扱うデスクトップアプリです。Tauriのローカルアプリとして、ユーザーの手元のファイル、ショートカット、画像、SQLiteデータを中心に動きます。

## 主要価値

- 散らばったゲームを一覧化し、起動しやすくする。
- プレイ状況やプレイ時間を記録し、積みゲー管理を助ける。
- ゲームごとのメモ、画像、スクリーンショットを近い場所で扱う。
- トレイ、グローバルショートカット、別ウィンドウなど、Windowsデスクトップアプリらしい導線を提供する。

## 主要ワークフロー

- ユーザーは手動追加、フォルダスキャン、ドラッグアンドドロップでゲームを登録する。
- 登録済みゲームはホーム、サイドバー、詳細画面、トレイメニューから探す。
- `Play` 実行時にプロセス監視とプレイ時間記録が始まる。
- プレイ中の一時停止、ショートカット起動、スクリーンショット取り込みを行う。
- 詳細画面でサムネイル、アイコン、メモ、プレイ状態、スクリーンショットを編集する。

## 改善時の優先軸

1. 既存ユーザーのデータを壊さないこと。
2. ゲーム登録、起動、記録の成功率を上げること。
3. 画面遷移や編集の待ち時間、迷い、誤操作を減らすこと。
4. Windows固有の体験を安定させること。
5. 将来の変更が小さく安全に入る構造に近づけること。
